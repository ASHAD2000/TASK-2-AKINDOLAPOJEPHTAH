# DATALAB Internship – Project 2

## Project Title

Exploratory Data Analysis (EDA) using Microsoft Excel

## Objective

To explore a cleaned e-commerce dataset using descriptive statistics and Pivot Tables to identify patterns, trends, and potential outliers.

## Tools Used

- Microsoft Excel 2021

## Analysis Performed

- Calculated descriptive statistics
- Identified the most ordered product
- Analyzed referral source performance
- Analyzed payment methods
- Identified potential outliers
- Summarized business insights

## Key Findings

- Dataset contains 1,200 transactions
- Printer recorded the highest number of orders (181)
- Instagram generated the highest referrals (259)
- Online payment was the preferred payment method (258)
- A few high-value transactions were identified as potential outliers
